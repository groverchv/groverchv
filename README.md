<img align="right" src="https://github.com/7oSkaaa/7oSkaaa/blob/main/Images/Right_Side.gif?raw=true" width="250px">

¡Hola! Soy **Desarrollador Full Stack** y estudiante de **Ingeniería de Sistemas (9no Semestre)** en la UAGRM. Me especializo en la arquitectura e implementación de plataformas web y móviles de alto rendimiento utilizando tecnologías modernas.

Mi enfoque principal es la integración de tecnologías disruptivas como **Inteligencia Artificial (Gemini, LLaMA)**, **Visión Computacional** y **Realidad Aumentada**, transformando requerimientos complejos en soluciones escalables.

* 🔭 **Proyectos destacados:** "CoachVirtual" (Visión Computacional) y Sistemas de Planificación Académica.
* 🌱 **Especialidad:** Spring Boot, React, Angular, Django y Flutter.
* 💬 **Enfoque:** Optimización de procesos operativos y arquitectura de software.
